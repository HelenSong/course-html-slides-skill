# 风格预设方案

当用户没有明确的配色偏好时，提供以下预设方案供选择。每个方案都经过教学场景优化——大屏可读性高、对比度达标、视觉层级清晰。

> 亲和力强，适合 K12、亲子教育、社区公益课

| Token | 值 | 说明 |
|---|---|---|
| `--base-bg` | `#F9F6EF` | 米白底色 |
| `--primary` | `#F66B31` | 品牌橙（主按钮、强调） |
| `--secondary` | `#4A99E9` | 科技蓝（第二强调） |
| `--accent-text` | `#1E1A18` | 正文黑 |
| `--surface` | `#FFFFFF` | 卡片白 |

- **字体**: `"PingFang SC", "Microsoft YaHei", system-ui, sans-serif`
- **圆角**: 大圆角（`16px` ~ `32px`），亲和活泼
- **渐变方向**: 橙色渐变 `135deg, #F66B31 → #FF9A5C`；蓝色渐变 `135deg, #4A99E9 → #7CB9F0`

## 自定义风格

如果以上预设都不合适，引导用户提供：

1. **主色**（`--primary`）：品牌色或偏好色
2. **辅色**（`--secondary`）：与主色互补或同系对比的第二色
3. **底色**（`--base-bg`）：浅色 or 深色底
4. **氛围关键词**（2-3个）：用于推导圆角、字体、间距等
   - 例：「活泼 可爱 圆润」→ 大圆角 + 圆体字
   - 例：「专业 严谨 商务」→ 小圆角 + 无衬线体

根据关键词自动推导完整设计系统参数。
